âž— Theoretical Foundations

MOND (Modified Newtonian Dynamics) proposes that the law of gravity changes at accelerations below a fundamental constant $a_0 \approx 1.2 \times 10^{-10} \text{ m/s}^2$.

The Quasi-Linear MOND (QUMOND) Relation

\[\mathbf{g} = \nu\left(\frac{|\mathbf{g}_N|}{a_0}\right) \mathbf{g}_N\]

This formulation provides a computationally efficient path to solve the non-linear Poisson equation [Milgrom, 2010].

Interpolation Function (\(\nu\))

\[\nu(y) = \frac{1}{2} + \sqrt{\frac{1}{4} + \frac{1}{y}}\]

Where \( y = |\mathbf{g}_N| / a_0 \). In the high-acceleration limit (\(y \gg 1\)), \(\nu \to 1\) (Newtonian). In the deep-MOND limit (\(y \ll 1\)), \(\nu \to 1/\sqrt{y}\).

To address observational challenges like the Bullet Cluster and Hubble Tension, the engine implements several advanced physical modules.

1. Galaxy Cluster Dynamics (The Neutrino Patch)

MOND under-predicts gravity in clusters by a factor of ~2 [Banik & Zhao, 2022]. We implement sterile neutrinos as a "Hot Dark Matter" component that pools in clusters but stays diffuse in galaxies.

typedef enum { TYPE_BARYON, TYPE_NEUTRINO } ParticleType; // Neutrinos are initialized with high thermal velocity (e.g., 1000 km/s) // This prevents them from collapsing on small scales but fixes the Cluster potential. if (p[i].type == TYPE_NEUTRINO) { p[i].vel = random_thermal_velocity(TEMPERATURE_NEUTRINO); }

2. The Bullet Cluster (SPH Hydrodynamics)

Simulating the separation of gas from gravitational peaks requires Smoothed Particle Hydrodynamics (SPH). Gas particles experience pressure; collisionless MOND particles do not.

// Apply repulsive pressure force between gas particles if (dist < h_smoothing) { Vector3 pressure_force = calculate_pressure(particles[i], particles[j]); apply_force(&particles[i], pressure_force); }

3. Comoving Cosmology (Expansion Drag)

To model large-scale structures and the KBC Void, we implement the comoving formulation of MOND with Hubble Drag.

\[\frac{d^2\mathbf{x}}{dt^2} + 2H(t)\frac{d\mathbf{x}}{dt} = -\frac{1}{a^3}\nabla\Phi_{MOND}\]
